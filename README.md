# 🤖 AI Social Media Agent

A powerful, Streamlit-based web application that uses **Llama 3.1 (via Groq)** to brainstorm, draft, evaluate, and schedule social media content.

## 🚀 Overview
Managing social media is time-consuming. This agent acts as a creative partner that understands platform-specific nuances for Instagram, LinkedIn, X (Twitter), and YouTube.

### ✨ Key Features
*   **Idea Generation:** Get 5 creative professional ideas based on any topic and tone.
*   **Platform Repurposing:** Instantly turn one idea into a Tweet, a LinkedIn post, or a YouTube Shorts script.
*   **Hook Generator:** Generates 8 attention-grabbing hooks for every idea.
*   **Strength Evaluator:** AI analyzes your draft and provides a score (0-100) with specific suggestions for improvement.
*   **Content Scheduler:** Built-in calendar logic to preview and export your posting plan to CSV.

## 🛠️ Tech Stack
*   **Frontend:** [Streamlit](https://streamlit.io/)
*   **LLM API:** [Groq Cloud](https://console.groq.com/) (Llama-3.1-8b-instant)
*   **Language:** Python 3.x
*   **Data:** Pandas for scheduling and CSV exports
